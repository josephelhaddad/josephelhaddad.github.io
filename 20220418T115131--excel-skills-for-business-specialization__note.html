1. Excel Skills for Business Essentials

1.1. Taking charge of excel

1.1.1. tachink charge of ecxel

  • when you are a self-taught, there are alot of gaps, and you don't know where those gaps are.
  • double click on the ribbon/bar will hide/show it.
  • on the top there is a quick access toolbar, you can add new shortcut like "quick ascending" from the dropdown or righ click on the tool and click add to toolbar

1.1.2. Navigating and selecting

  • use Ctrl+Home / Ctrl+End and PgUp / PgDn to navigate
  • if you want to select multiple cells, click on top left cell, hold Shift, and click no the buttom right cell.
  • if you want to select the whole table, click on Ctrl+a

1.1.3. View option

  • Split
    • in View => Window => Split (Alt+w+s)
    • click in the middle of your table and click on split, it will split the screen to four.
    • double click on the middle line to detele it, and you can now look at diffirent positions of the table at once:
    • Example: Split table to show top and buttom at the same time
  • Freeze panes
    • in View => Window => Freeze panes (Alt+w+f+f)
    • make the header (row) stick while scrooling up or down, and/or make columns stick while scrolling left and right.
    • Highliting a cell, and then clicking on freeze panes, it will freeze the rows above it and the columns in the left of it.
  • Switch windows
    • in View => Window => switch windows
    • if you open multiple excel files, you can switch between them with Ctrl+F6

1.1.4. Data entry, data types, editing and deletign

  • when entring a datum in a cell, press Tab to jump to the right or Ret to jump down.
  • to delete the datum in cell, press Del.
    • don't use the Delete in the ribbon as it remove the cell it self and shift all the data.
  • if you modify a cell and you don't want the change to take place, press Esc

1.1.5. Fill Handle

  • It's a way to quickly fill cells with copy, series, or patterns
  • if a cell have a datum, click in the buttom right corner of the cell and drag it to the desire cell.
  • Base on the type of the data, you can change the fill option, by clicking on the auto fill option icon in the burrom right.
  • if you want to re triger the auto fill option icon of a column, double ckick on the buttom right corner.
  • If there is a patterns and you want the autofill to go base on it, you have to select multiple cells
  • If the height is already predefine, you can double click in the bottom right corner of a cell to triger the autofill
  • Flash Fill is pattern matcher tool, it is based on the selected cell and the neiboring column pattern.

1.2. Performing Calculations

1.2.1. Formulas

  • Formulas are created with equal sign '=' plus cells and opperations
    • Example: =B4+B5+B6

1.2.2. Formula in conetcx

  • you can use the formulas with the fill handle for quick calculation

1.2.3. Functions: SUM

  • Sum are function that add up values in a range of cells.
  • instead of doing =B4+B5+B6, you do =SUM(B4:B6).
  • you can click on autosum located in Home => Editing => Sum, (Alt+u+s), to sum either all cells above or on the left of the cell selected.
  • and you can use fill handle

1.2.4. Functions: Average/Min/Max

  • same as above, just insert the function in a cell, select the data you want to be processed and click enter
  • Home => Editing => Sum(more functions), (Alt+u+f)
  • for average is (Alt+u+a), for min (Alt+u+i), and for max (Alt+u+m)
  • if some rown have a diffirent formating and you use the fill handle, drop down and chose "fill without formating".

1.2.5. Abselute cell reffrencing

  • when you are using a function and planing to use fill handle afterwords, if you are using a fixed cell in your calculation you need to add dollar sign to your cell id.
    • Example: =B4*$J$1

1.2.6. Caluculation accros cell

  • you can reffer to a value of a cell in another sheet by typing "=" and click on the cell in the sheet that you desire.
    • Example: ='Sales 2015'!F4, =SUM('Pay Details'!G7:G9)
    • after you select the sell in the other sheet click enter, do not turn back to the original sheet.
  • do switsh between sheet quickly use Ctrl+PgUp/PgDn

1.3. Formatting Spreadsheets

1.3.1. Font formating

  • Home => Font
  • you can change font size to make heading
  • add line, boldness, change text color, change cell color…

1.3.2. Borders

  • Home => Font
  • if you have a big header, select multiple cells and add a border to them
  • if tou want to remove border select no border

1.3.3. Alignment

  • Home => Alingment
  • You can align text horizontaly or verticaly in a cell.
  • Merge & center helps you make a header centered accross multiple cells by merging them.
  • Wrap text wrap long text into multiple lines, you can apply horizental and vertcal alignmetn

1.3.4. Format painter

  • Home => Clipboard => Format painter
  • you can copy the text formating from a cell/cells to another/others with Format painter.

1.3.5. Clear

  • Home => Editing => Clear
  • one of its role is to remove the formating from the cells.

1.3.6. Formating number

  • Home => Number
  • You can format the number in a cell to match its meaning, like currency, date, percentage…
  • Accounting number formating (Alt+h+an), allow you to quickly format cells to have number with two digits at the end and a currency symbol at the left.

1.3.7. Style and themes

  • Home => Style => Cell Styles (Alt+h+j)
  • To sellect all filled cells click Ctrl+Shift+Arrows
  • Page Layout => Themes (Alt+p+th)
  • You can only apply/save a theme if all the cells were formated using Cell Style.

1.4. Workign with data

1.4.1. Managing rows and columsn

  • you can delete, insert, and hide rows and columns

1.4.2. Find and replace

  • Home => Editing => Find & Select (Alt+h+fd)

1.4.3. Filtering

  • Data => Sort and filter => Filter (Alt+a+t)
  • On the right of any colums that have filter you can select more option base on the data type, you can have additional filters for dates, numbers, text.
  • you can also creal the filter in (Alt+a+c)
  • There is additional filter option for numeric data called "Top 10…"

1.4.4. Sort

  • Data => Sort and filter => Sort (Alt+a+ss)
  • Click on a cell in your table and then click on Sort button, chose that your data have a header, and then sort base on culumn and chose the order
  • you can add a sort level for multiple orders.

1.4.5. Conditional formating

  • Home => Styles => conditional formating (Alt+h+l)
  • change the colum style base on the data in the cell it self
  • you can clear cell rules

1.4.6. Accessibility

  • Press the Alt key to see shortcut if you dont have a mouse or you want to speed your process

1.5. Printing spreadsheet

1.5.1. Print preview

  • sometimes when you highlight the whole row you get more page to print than desired, when that happend you can select the data you want to print and chose "print selected" in the settings.

1.5.2. Oriontation, margin, and scale

  • the first tool that help accessing is page layout
  • to make the page fit:
    • You can change the oriontation in the page layout.
    • You can set the margin to narrow
      • you can go to custom marging and set center on page to horizontally.
    • You can Scale to fit base on width or height

1.5.3. Page breaks

  • Page layout => Page setup => Breaks
  • if you want to break the printed table on a row use breaks
    • Example: you have sorted the table base on courtys and you dont want for the buttom rown of a country be next to the top rows of another.
  • In the page break preview you can drag the line where you want to break the pages

1.5.4. Print titles

  • Page layout => print titles
  • If you want to keep the header and an ID column across all the printed pages, chose rows to repeat at the top and columns to repeat at the left by selecting them.
  • you can change the page printing order to be from right to buttom instead of buttom to right.

1.5.5. Header and footer

  • in page preview you can see a message that say "add a header", in a row that is devided by 3 sections
  • if you click on one of the sections you can see a new tab in the ribbon called design you can add element from header and footer elemnts
  • example: page 1 of 21 &[Page] of &[Pages]

2. Excel Skills for Business Intermediat

2.1. Working with multiple sheet and workbook

2.1.1. Multiple worksheeet

  • you cat rename and change the color of a sheet
  • if you want to make a change in a single cell across all sheets.
    • select all sheet with shift or Ctrl
    • click on the cell you want to modify
    • click on one of the sheet to cancel edit mode
  • you can hide and unhide sheets

2.1.2. 3D formulas

  • Same in calculating across cell, but its on multiple sheet in the same cell position
  • if you want to add cells from multiple sheet in the same position:
    • start with "=" and type a function
    • click on the first sheet and click on the cell
    • hold shift and click on the last sheet
    • and press enter
    • Result: =SUM(sean:carlos!C7), =SUM(Sheet1:Sheetn!C7)
  • If you change the order of the sheets and make one if them comes in last instead of the current one you will lose the data
  • You can copy and paste and excel will adjust the formula or funtion base on the paste location, and you can use the fill handle.

2.1.3. Linking workbook

  • same with sheets we can link workbooks as long as they wont be moved or renames
  • Open all workbook and arange them all
  • click on the cell and type "="
  • click on the workbook and the cell you want and press enter
    • the cell will be in the abselure refferencing, type F4 multiple times if you are planing to use the fill handle and the copy.

2.1.4. Consulidation by position

  • consulidate by position merge all data from diffirent workbook if the structure is the same
  • is only copy the data, there for it is no dependent on all workbook to be in the same path, but also, it does not update automaticly ones a value is updated in a workbook.
  • to consalidate by position:
    • open all workbooks
    • click on the top left cell that you want to fill with data
    • click on Data => data tools => consulidate
    • chose the function
    • click on any workbook and select the area wheer data is starting from top left
    • you can press add if needed
    • Press Enter

2.1.5. Consulidation by reffreance

  • mostly used if the rows/column that represent a category arent inline with each other in all workbook.
  • to consalidate by category:
    • open all workbooks
    • click on the top left cell that you want to fill with data
    • click on Data => data tools => consulidate
    • chose the function
    • click on any workbook and select the area where the refferance is starting from top left and select all the data
    • you can press add if needed
    • Use label in: left column
    • Press Enter

2.2. Text & Date functions

  • a function can have mulitple argument seperated with ","
    • Example: =sum(b2:d2,h2,average(e2:g2))

2.2.1. Combining text

  • =CONCAT(cell,"added text or space",cell)
  • Example:
    • =CONCAT(B4," ",C4)
    • =CONCAT(C4,".",B4,"@gmail.com")

2.2.2. Changing text case

  • text can be changed to:
    • UPPER: =UPPER()
    • lower: =LOWER()
    • Proper: =PROPER()
  • Example:
    • =PROPER(CONCAT(B4," ",C4))
    • =LOWER(CONCAT(C4,".",B4,"@gmail.com"))

2.2.3. Extracting text

  • You can extract text of a cell starting from the left, right, or middle, by specifiying a number.
  • Example:
    • =LEFT(A2,2)
    • =Right(A2,4)
    • =MID(K4,4,FIND(" ",K4)-FIND("-",K4)-1)
  • The middle can be tricky, it take the text cell, the start number, the number of text lenght. that why we need a find function

2.2.4. Find Text

  • it is like the split function in python.
  • =FIND(" ",Cell)
  • Exapmle:
    • =FIND(" ",K4)
    • =FIND(" ",K4)-FIND("-",K4)-1, this return the lenght of the text that can be used in MID function

2.2.5. Time

  • =NOW(), return the current date and time
  • =TODAY(), return the current date
  • =YEARFRAC(F4,TODAY()), turn any number into years

2.3. Named ranges

2.3.1. introduction

  • you can change a cell name in the Name box next to the formula bar
  • Naming a cenn makes is an abselute cell when its reffrenced.
  • Example: change a cell from P1 to Pension_Rate. so instead of doing =N4*$P$1, we now write =N4*Pension_Rate.

2.3.2. Creating names ranges

  • scope is telling excel where the naming should be, whether across all workbook or only in a sheet.
  • there are 3 ways to create a name for columns:
    • Name box: select cells that contain data of a colum and name it in the name box
    • Formulas => Define names: select the dat aand click on define name
    • Formulas => Create from selection: select the header and the data and click on Create from formulas
      • Click on the header
      • Ctrl+Shift+down
      • Or click somewhere in the data base and type Ctrl+a
      • Ctrl+Shift+3

2.3.3. Managing names ranges

  • Formulas => Define names => Name Manager
  • you can edit, delete, and add new name range.
  • The scope cannot be changes

2.3.4. Name ranges in formulas

  • Ctrl+Enter to finish the formula ans stay in the same cell to apply fill handle
  • F3 to show all name range
  • Example: =AVERAGE(Annual_Salary)

2.3.5. Apply name

  • Formulas => Define names => Define names => Apply name
  • apply name is used to replace a range, in a premade formula, with a name range.

2.4. Summirasing data

2.4.1. Count function

  • =COUNT(name_range_colum): count numeric cells in a column.
  • =COUNTA(name_range_colum): count numeric and alphabetic cells in a column.
  • =COUNTBLANK(name_range_colum): count blank cells in a column.

2.4.2. Counting with condition

  • the IFS gives you the ability to have multiple conditions
  • =COUNTIFS(name_range_colum, condition), count the number of cells in a column that meet the condition
  • if the condition is a cell or a number, you can pass it normaly, however, if the condition is text or expresstion they need to be within "".
  • Example:
    • =COUNTIFS(Customer_Type,A11)
    • =COUNTIFS(Order_Quantity,">40")

2.4.3. Sum ifs

  • sum cell base on conditions
  • In abselute sell refference, there is 3 type:
    • $A$1: dollor infront of the row and colum reffrence mean that you will use the same value of the cell A1 when using the fill handle
    • $A1: dollar infront of the column refferance mean that you will only use the cells in the A colum when the fill handle is dragged to the right.
    • A$1: dollar infron of the row refferance mean that you will only use the cells in the 1st row when the filll handle is dragged down.
  • Example: getting 2 condition
    • =SUMIFS(Total,Account_Manager,$A21,Order_Year,C$20)

2.4.4. Sparlines

  • are graph that show a trend in a range of cells, and can be fit in a single cell
  • Insert => Sparklines
  • you cannot double click to activate the fill handle with sparklines, you need to drag manually.
  • if you want to remove a sparklies you need to use the clear option in the sparkline new tab.

2.4.5. Advanced charting

  • sometimes the chart isent as you will want it to be, if that is the case you can twaek the chart from the chart design.
  • Chart design => Data => Switch row/column
    • use it if the categories and the x-axis need to swap.
  • Chart design => Data => Select data
    • use it if excel fail to select the rows and columns, click edit and chose a range.
    • also if you want to hide a centain rows or columns.
  • Chart design => Type => Change chart type
    • if you have on colum that is big, you can use combo chart, and give that big colum its own chart that is merged with the original one but with separate y-axis

2.4.6. Trendlines

  • you can click line in the chart and click on the + or add element and chose trand line

2.5. Tables

2.5.1. Creating tables

  • If the data is changing and growing it is better to use tables since the ranges grow with every addition of records
  • Click anywere on the data and type Ctrl+t

2.5.2. Working with tables

  • If you want to use Ctrl+shift+arrow to select a row or a column, it wont work if there is a blank cell in the way.
  • In tables, hover on the left of the first cell and you ll see an arrow that if clicked on, it will select all the data in the left.
  • In tables, you can select the whole table either by typing Ctrl+a, type it again to select the header as well. of hover on the top left of the first colum until you see a diagonal arrow.
  • To remove duplucates.
    • Table design => Tools => Remove duplicates
  • When you scrol down the header of table stay fix, so there is no need for freezing panes

2.5.3. Sort, filter and Total

  • sort and filter are the same
  • Total is a row that can be added to the end of the table and alows you to preform calculation on the columns wheer you want the sum, average, std ..

2.5.4. Automation

  • Ctrl++ create a new row
  • Pressing Tab in the last cell of the last row will create a new row
  • Ctrl+; to enter today date
  • The table will update the name ranges
  • Making fomulas result in applying it emediatly without the fill handle
    • Example: =[@[Annual Salary]]-[@Pension]

2.5.5. Subtotal

  • that data must be converted to range
  • Data => Outline => Subtotal
  • this function let you chose a two or more categorical column and numerical column to apply a calculation to.

2.6. Pivot table

2.6.1. Creating and modifying

  • Pivot tabels help us make summery tables much faster
  • Creating pivot table
    • create a table
    • Insert => pivot table
  • pivot table aims to summarize a categorical column with a numerical column
  • the columns of the table are called field
  • the fiekd can be draged into 4 areas
    • Filters
    • Columns
    • Rows
    • Values
  • once the data is updated click refresh

2.6.2. Value field setting

  • you can change how the value area calculate the values
  • chose a value cell within the pivot table nad right click
  • summurize value by let you chose to apply sum, count, avg, and more
  • shose value as let you change the format to percentage, eather base on the grand total only, comulns, or rowns
  • If you put two fields in the pivot table row area you get a subtotal table

2.6.3. Sort and filter

  • if you have a field in columns area you can ungrop and group them
  • if there are dates you can group them frmo Data => outline for Quarter analysis
  • you can eother filter the columns that is already in the fiels or add a new column in the filter area and filter

2.6.4. Reporting filter page

  • you cann add a field in filter area
  • If you add multiple choise in the filter the result in the pivot table will show as sum
  • Pivot table tools => Analyse => Pivot table => Option(dropdown) => show report filter pages
  • double clicking in pivot table cell makes a new sheet that contain the data that was used to give the result calculated base on the condition you set. (it is just a copy, it wont update when the origina table update)

2.6.5. Pivot chart

  • Pivot table tools => Pivot chart
  • you add a chart, change columns and rows, apply filter within the chart…

2.6.6. Pivot slices

  • Pivot table analyse => Filter => Insert Slicer
  • Click on pivot table or pivot chart to activate slicer button
  • Slicer brings a list of all the filter that can be applyed to a table or chart base on selected columns.
  • Best used for a dashboard where certain people would like to filter the report to see theire desire graphs
  • You can connect the slicer with an other pivot chart by clicking on the slicer Slicer => Report connection

    • invest in savings= you can invest your money in a bank, depending on how long you chose to do it, you can lend the bank your money, each month for 25 years, number of period is 25*12=300. or each year for 25 years, in that case the number of period will be 25.

  • Inderect functions gives you the datum in a cell if you give it a text that refere to either a cell position or name range.
  • it is mostly useful if you have a drop down value (text) and you want to look for value base on it.
  • you can spesify a column ("G") and use the match function to return the row number of the cell that contain an element in dropdown menu.
    • Example:
      • =INDIRECT("G"&MATCH(C3,F1:F14,0))
      • =INDEX(G2:G14,MATCH(C3,County_List,0))
  • If you have a dropdown cell with value that refer to a name range, you can use the indirect function to retrieve the whole array of data and use it with a arhetmitic function.
    • Example:
      • =SUM(INDIRECT(C5)), C5 refere to a cell that contain a text similar to a name range
  • If you have a dropdown cell with value that refer to a neme range, and you want to average the rop 3 values in that range, use large function, with row fucntion
    • Example:
      • {=AVERAGE(LARGE(INDIRECT(C5),ROW(1:3)))}, use Ctrl+Shift+RET
  • If you have a dropdown cell with value that refer to a name range, and you want that cell to determin what it will show in another dropdown list, use data validation with indirect. but you need to prepare name ranged beforehand
  • Steps:
    • Prepare a sheet with neccesary data, remove dupes in the more spesific column
    • use names range to group the data you wwant to use in dependent drowdown menu
    • in the nex dropdown cell, use list data validation ,and inderect fuction
      • Example:
        • =INDIRECT($C$3)
