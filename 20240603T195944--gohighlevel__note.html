2.8. Missed call text back

  • The benefits of having missed call back automation instead of GHL default one:
    • Check if the customer who called is serious by sending an SMS and awaiting a reply.
    • Automatically add a lead's name if they answer the SMS where we ask them how we can address them.
    • Customize who will get notified.
  • Folder name: C02. Lead Management
  • Workflow name: C02.1 Missed Call Text Back
  • Trigger: Call Status
    • Filters:
      • Call direction: incoming
      • Call status: busy, canceled, voicemail, no-answer
  • Actions:
    1. Action: Update contact field
      • Fields:
        • Contact source: "Phone incoming call"
    2. Action: Send SMS Ask them about their name so that it will be captured in the next actions; Example: "Sorry we ;issed your callm we will reach out to you shortly on this number, Could I please have your name so I know who we will be addressing?"
    3. Action: Wait
      • Wait for: Contact reply
      • Reply to: SMS
      • Timeout: five minutes
    4. Action: If/else
      • Segments:
        • Contact replied: True
          • Actions:
            1. Action: Update contact field
              • Fields:
                • First name: {{message.body}} This will capture any name found in the SMS respond.
            2. Action: Send SMS "Thank you, we will reach out shortly"
            3. Action: Send internal notification
              • Type of notification: Email, Notification, SMS Use custom values like: {{001. Email that notifications/leads are sent to}}
              • Message: you have a missed call from {{contact.name}} - {{contact.phone}}
            4. Action: Create or Update Opportunity
              • In pipeline: Lead
              • In pipeline stage: Missed Call
              • Allow opportunity to move to any previous stage in the pipeline.
            5. Action: Manual Action to call This will add the number to a list for you to manually call with a click of a button in Conversation \(\rightarrow\) Manual Actions Note: if you have call recording on, upon calling the customer, a voice will say "This call will be recorded…", this may repel the customer
        • Contact replied: False
          • Actions:
            1. Action: Send internal notification
              • Type of notification: Email, Notification, SMS Use custom values like: {{001. Email that notifications/leads are sent to}}
              • Message: you have a missed call from {{contact.name}} - {{contact.phone}}
            2. Action: Create or Update Opportunity
              • In pipeline: Lead
              • In pipeline stage: Missed Call
              • Allow opportunity to move to any previous stage in the pipeline.
            3. Action: Manual Action to call This will add the number to a list for you to manually call with a click of a button in Conversation \(\rightarrow\) Manual Actions Note: if you have call recording on, upon calling the customer, a voice will say "This call will be recorded…", this may repel the customer
  • Settings:
    • Check "Allow for re-entry"
    • Check "Mark as read"
